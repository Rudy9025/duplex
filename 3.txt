To implement a function in Angular that downloads data as a CSV (or Excel) file only if the data has length > 0, you can use the following approach:

1. Add the ExportToCSV method in your component (exclusion-rule.component.ts):

ExportToCSV() {
  if (!this.excluded || this.excluded.length === 0) {
    console.warn('No data to export');
    return;
  }

  const csvRows = [];
  const headers = Object.keys(this.excluded[0]);
  csvRows.push(headers.join(','));

  for (const row of this.excluded) {
    const values = headers.map(header => {
      const escaped = ('' + row[header]).replace(/"/g, '\\"');
      return `"${escaped}"`;
    });
    csvRows.push(values.join(','));
  }

  const csvData = csvRows.join('\n');
  const blob = new Blob([csvData], { type: 'text/csv' });
  const url = window.URL.createObjectURL(blob);
  const a = document.createElement('a');
  a.setAttribute('href', url);
  a.setAttribute('download', 'excluded_data.csv');
  document.body.appendChild(a);
  a.click();
  document.body.removeChild(a);
}

2. HTML already has the click binding:

<button (click)="ExportToCSV()">Export</button>

Explanation:
	•	Checks if this.excluded is non-empty.
	•	Extracts headers from the first object.
	•	Escapes and joins each row.
	•	Triggers a download with a temporary anchor tag.
